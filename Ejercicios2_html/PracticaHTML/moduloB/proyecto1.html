¿Qué es una página web?

Las páginas web pueden ser nombradas de muchas maneras: página electrónica, página digital, ciberpágina, sitio web, portal web, etc. Todas ellas se refieren a un documento escrito generalmente en formato HTML o XHTML en el cual hay una serie de instrucciones para mostar información. Esta información puede ser de tipo texto, imágenes audios y videos. Los documentos HTML pueden ser acompañados por otros archivos como hojas de estilos (CSS) y archivos de Script (Javascript). Las páginas web pueden ser accedidas desde un navegador web como Chrome, Firefox o Edge y pueden ser enlazadas mediate enlaces de hipertexto.
